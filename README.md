# SmartExpense – Offline Expense Tracker (Android)

SmartExpense is an **offline-first Android expense tracker** that allows users to record daily expenses, organize them into categories, and view clear spending summaries — all without requiring an internet connection.

This project is built using **modern Android development practices** and is designed with scalability in mind.

---

## Features

- Add, edit, and delete expenses
- Categorize expenses (Food, Travel, Shopping, etc.)
- View daily and monthly spending totals
- Fully offline using local persistence
- Clean and modern UI with Jetpack Compose
- Scalable architecture ready for future online sync

---

## Tech Stack

- **Language:** Kotlin  
- **UI:** Jetpack Compose, Material 3  
- **Architecture:** MVVM  
- **Database:** Room (SQLite)  
- **Async & State:** Coroutines, Flow  
- **Build:** Gradle (Kotlin DSL)

## Getting Started

2. Open the project in **Android Studio**
3. Let Gradle sync complete
4. Run the app on an emulator or physical device
